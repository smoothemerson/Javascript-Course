Math

Math é um objeto embutido que tem propriedades e métodos para constantes e funções matemáticas.

Para usar a função Math, você precisa declarar a variável e depois usá-la

Exemplo

let x = 3;

x = Math.round(x)

Código

round: arredonda o número

floor: arredonda o número para baixo

ceil: arredonda o número para cima

pow: coloca expoente no número

sqrt: coloca raiz quadrada no número

abs: mostra o quão longe do 0, o número está

max: mostra qual é o maior número entre um conjunto de variáveis

min: mostra qual é o menor número entre um conjunto de variáveis

PI: mostra o valor de PI
