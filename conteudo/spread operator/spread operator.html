Spread operator

Respresentado por '...'

Permite que uma repeticão como um array ou uma string seja expandida em lugares onde zero ou mais argumentos são esperados (desempacota os elementos)

É como abrir uma caixa e separar as coisas que estão dentro

O operador de propagação, também conhecido como spread operator, é representado por três pontos consecutivos: ....

Ele é usado para desdobrar (spread) elementos de uma estrutura iterável, como um array ou um objeto, em lugares onde vários elementos (ou argumentos) são esperados.

Existem duas formas principais de utilização do spread operator:

  1. Para arrays:

    O spread operator permite copiar os elementos de um array para outro array, de forma rápida e concisa.

  2. Para objetos (strings):

    O spread operator também pode ser usado para mesclar propriedades de objetos em um novo objeto.
