Data Object

É usado para trabalhar com datas e tempos


Um objeto Date é usado para trabalhar com datas e horários. Ele representa um ponto específico no tempo e fornece vários métodos para manipular e exibir datas.

Alguns dos métodos mais usados do objeto Date em JavaScript são:

  1. getDate(): Retorna o dia do mês (entre 1 e 31) da data especificada.
  2. getMonth(): Retorna o mês (entre 0 e 11) da data especificada. (Obs: Janeiro é representado por 0 e dezembro por 11).
  3. getFullYear(): Retorna o ano completo (com 4 dígitos) da data especificada.
  4. getHours(): Retorna as horas (entre 0 e 23) da data especificada.
  5. getMinutes(): Retorna os minutos (entre 0 e 59) da data especificada.
  6. getMinutes(): Retorna os minutos (entre 0 e 59) da data especificada.
  7. toDateString(): Retorna uma string representando a data (sem incluir informações de hora).
  8. toLocaleString(): Retorna uma string representando a data e hora formatadas de acordo com as configurações regionais do sistema.
