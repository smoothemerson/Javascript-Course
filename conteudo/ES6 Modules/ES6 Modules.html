ES6 Modules

A ideia por trás de um módulo é que é um arquivo de um código reutilizável

Nós podemos importar seções de códigos pré-escritos para usar quando quisermos

ótima para qualquer valor ou funções

Ajuda a manter seu código mais reutilizável e sustentável

Pense em módulos como se fossem capítulos de um livro


Os módulos ES6 permitem exportar e importar funções, objetos, classes ou valores de um arquivo para outro. Com essa abordagem, você pode dividir seu código em módulos independentes, onde cada módulo é responsável por uma tarefa específica. Isso ajuda a melhorar a modularidade, a legibilidade e a manutenção do código.
