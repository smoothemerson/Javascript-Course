Array.filter()

Cria um novo array com todos os elementos que passam por um teste providenciado por uma função


O método array.filter() é uma função de iteração em JavaScript que permite filtrar elementos de um array com base em uma condição especificada e criar um novo array contendo apenas os elementos que atendem a essa condição.

Aqui estão as principais características do array.filter():

  • array.filter(callback): O método filter() é chamado no array que desejamos filtrar e recebe uma função de callback como argumento.
  • callback: A função de callback é executada para cada elemento do array. Ela recebe um parâmetro que representa o elemento atual e retorna true ou false com base em uma condição.
  • Retorna um novo array: O filter() retorna um novo array contendo apenas os elementos do array original que atendem à condição definida na função de callback.
  • Utilização: A função de callback pode ser uma função anônima ou uma função nomeada definida anteriormente. Dentro da função de callback, você define uma condição que determina se um elemento deve ser incluído ou não no novo array resultante.

O array.filter() é útil quando você precisa filtrar elementos de um array com base em critérios específicos. Ele permite que você crie um novo array contendo apenas os elementos que correspondem à condição definida na função de callback.
