Rest parameters

Representa um número indefinido de parâmetros, empacota os argumentos em um array

O parâmetro rest, também conhecido como rest parameter, é uma funcionalidade que permite capturar um número indefinido de argumentos em uma função.

Ele é representado pelo operador de três pontos consecutivos ... seguido por um nome de parâmetro.

O parâmetro rest é usado quando você deseja lidar com uma quantidade variável de argumentos que são passados para uma função.

Em vez de definir explicitamente cada argumento individualmente, você pode usar o parâmetro rest para capturar todos os argumentos restantes em um array.
