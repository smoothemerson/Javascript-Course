Array.map()

Executa uma função callback providenciada para cada elemento do array e cria um novo array


O método array.map() é usado para iterar sobre os elementos de um array e criar um novo array com base em uma transformação aplicada a cada elemento.

Ao chamar array.map(callback), o método percorre cada elemento do array e executa uma função de callback para cada elemento. A função de callback recebe como argumento o elemento atual e pode retornar um novo valor que será adicionado ao novo array resultante.

O novo array retornado pelo map() terá o mesmo comprimento do array original, pois cada elemento é mapeado para um elemento correspondente no novo array, com base na transformação definida na função de callback.
