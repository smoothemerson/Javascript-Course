Variáveis

Uma variável se comporta como se um valor estivesse dentro dela

Etapas para declarar uma variável:

  1. Declaração (var, let, const)
  2. Assinatura( "=" é o operador de assinatura)

Tipos de variáveis:

  • Strings: é um texto
  • Números
  • Valores Booleanos: True ou False
