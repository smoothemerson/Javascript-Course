Getters & Setters

Getters = liga a propriedade de um objeto para uma função quando essa propriedade é acessada

Setters = liga a propriedade de um objeto para uma função quando essa propriedade receber algum valor


Getters

Os "getters" são métodos especiais que permitem acessar os valores de propriedades de um objeto. Eles fornecem uma maneira de recuperar (ou obter) o valor de uma propriedade específica de um objeto, como se fossem uma propriedade regular, mas com a lógica personalizada.

Os getters são úteis quando você deseja adicionar lógica personalizada para recuperar os valores das propriedades de um objeto, como formatar, filtrar ou calcular um valor antes de retorná-lo. Eles permitem um controle mais refinado sobre o acesso aos dados de um objeto.

Setters

Os "setters" são métodos especiais que permitem atribuir valores a propriedades de um objeto. Eles fornecem uma maneira de definir (ou alterar) o valor de uma propriedade específica de um objeto, como se estivessem atribuindo um valor a uma variável, mas com a lógica personalizada.

Os setters são úteis quando você deseja adicionar lógica personalizada para validar ou processar os valores atribuídos a propriedades de um objeto. Eles permitem um controle mais refinado sobre a atribuição de valores e podem ser usados para garantir que as propriedades sejam configuradas corretamente.
