DOM intro

DOM = Modelo de Objeto de Documento

Uma interface para mudar o conteúdo de uma página

É uma API = Aplicação de Interface de Programação


O DOM (Document Object Model) é uma interface de programação que representa a estrutura de um documento HTML como uma árvore de objetos. Ele fornece uma representação estruturada do conteúdo de um documento, permitindo que os programadores acessem, manipulem e atualizem dinamicamente os elementos e os dados presentes na página web.

O DOM permite que você acesse e manipule os elementos do documento, adicionando, removendo ou alterando elementos, atributos e conteúdo. Ele fornece métodos e propriedades que permitem interagir com os elementos da página, como buscar elementos por id, classe ou tag, modificar estilos, adicionar ou remover classes, manipular eventos, entre outros.
