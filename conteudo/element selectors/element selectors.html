Element selectors em JavaScript são métodos que permitem selecionar elementos HTML específicos em uma página web para manipulação e interação através do código JavaScript. Eles permitem que você encontre elementos com base em seus atributos, classes, identificadores (IDs) ou tags, permitindo acesso e manipulação direta desses elementos.

Segue a lista de métodos:

  • getElementById
  • getElementsByClassName
  • getElementsByTagName
  • querySelector
  • querySelectorAll
