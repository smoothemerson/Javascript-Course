AddEventListener()

Podemos linkar um evento para uma função

.addEventListener(event, function, useCapture)

Você pode adicionar vários manipuladores de eventos para um elemento

Mesmo se o mesmo evento invoca diferentes funções


O addEventListener() é um método que permite que você defina uma função para ser executada quando um evento específico ocorre em um objeto, como um elemento HTML. Ele é usado para "ouvir" eventos e executar ações correspondentes quando o evento ocorre.

Um evento pode ser qualquer ação que acontece em um elemento, como um clique de mouse, uma tecla pressionada, uma mudança de valor em um campo de formulário, entre outros.
