Asynchronous

Código Síncrono

Funciona em uma sequência ordenada, passo a passo, instruções lineares (comece agora, termine agora)

  • No código síncrono, as instruções são executadas em sequência, uma após a outra, de forma bloqueante.
  • Cada instrução é executada e concluída antes de passar para a próxima instrução.
  • Durante a execução de uma instrução, o programa aguarda até que essa instrução seja concluída antes deprosseguir.
  • O fluxo de execução é linear e previsível.

Código Assíncrono

Funciona fora de sequência (comece agora, termine algum tempo depois)

  • No código assíncrono, as instruções não são executadas em sequência linear.
  • As instruções podem ser agendadas para execução futura ou para serem executadas em paralelo.
  • A execução de uma instrução não bloqueia o programa, permitindo que outras instruções sejam executadas simultaneamente ou enquanto aguardam a conclusão de operações assíncronas.
  • O fluxo de execução pode ser interrompido e retomado conforme necessário.
