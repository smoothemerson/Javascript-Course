Console.time()

Começa um timer que você pode usar

Mede quanto tempo uma operação leva

Podemos nomear cada timer


O método console.time() é uma função em JavaScript que permite medir o tempo de execução de um bloco de código específico. Ele cria um temporizador no console do navegador que começa a contar assim que é chamado e termina quando o método console.timeEnd() correspondente é chamado com o mesmo rótulo.
