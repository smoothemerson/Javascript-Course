Array

Array = pense que é uma variável que pode guarda vários valores

Existem alguns atributos que podemos atribuir para um array como:

  • push: adiciona um elemento
  • pop: remove o último elemento
  • unshift: adiciona um elemento no começo
  • shift: remove o primeiro elemento
  • length: conta quantos elemento há no array
  • indexOf: mostra qual elemento está naquele índice
