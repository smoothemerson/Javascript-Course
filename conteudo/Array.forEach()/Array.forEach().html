Array.forEach()

Executa uma função array providenciada para cada elemento do array


O método array.forEach() é uma função de iteração em JavaScript que permite percorrer os elementos de um array e executar uma função de callback para cada elemento. Ele segue as seguintes características:

  • array.forEach(callback): O método forEach() é chamado no array que desejamos percorrer e recebe uma função de callback como argumento.
  • callback: A função de callback é executada para cada elemento do array. Ela recebe um parâmetro que representa o elemento atual do array durante a iteração.
  • Utilização: A função de callback pode ser uma função anônima ou uma função nomeada. Dentro da função de callback, você pode realizar qualquer ação desejada com base no elemento atual, como exibir o valor, realizar cálculos, modificar o array original, entre outros.
  • Não retorna um novo array: O forEach() não retorna um novo array. É utilizado principalmente para executar ações em cada elemento do array, sem modificar o próprio array.

O array.forEach() é útil quando você precisa percorrer todos os elementos de um array e realizar uma operação em cada um deles, como exibir, processar ou atualizar os valores. Ele simplifica o processo de iteração sobre um array e execução de ações personalizadas em cada elemento.
