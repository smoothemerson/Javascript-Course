"show" e "hide" são termos comumente usados para manipular a visibilidade de elementos HTML na página. "Show" refere-se a tornar um elemento visível, enquanto "hide" refere-se a torná-lo invisível, ocultando-o da exibição.

A manipulação da visibilidade de elementos HTML é geralmente feita através da propriedade display do CSS. A propriedade display permite controlar como um elemento é renderizado na página. Para mostrar um elemento, você precisa definir sua propriedade display como um valor que o torne visível, como "block" ou "inline". Para ocultar um elemento, você precisa definir sua propriedade display como "none", que o torna invisível.
