Métodos úteis para Strings

Índice é como se fosse a posição, mas para o computador é a posição -1, o computador começa a contar as posições pelo 0

Exemplo: no texto "teclado" o 4° índice é a letra "a"

Length: contará quantos caracteres



CharAt: digite um texto aleatório e coloque um índice que você deseja



indexOf: digite um texto e coloque uma letra que deseje encontrar seu primeiro índice



lastIndexof:digite um texto e coloque uma letra que deseje encontrar seu último índice



trim: removerá todas os espaços nas extremidades



toLocaleUpperCase: colocará todas as letras maiúsculas



toLocaleLowerCase: colocará todas as letras minúsculas



replaceAll: digite seu nome completo que esse comando substituirá todos os espaços para "-"
